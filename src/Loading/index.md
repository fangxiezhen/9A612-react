# Loading

在网络较慢或数据较多时，表示数据正在加载的状态

## 图标加载

加载过程中只有图标显示。适用于打开页面或操作完成后模块内等待刷新的加载场景。

## 文字加载

加载过程中有文字显示。适用于打开页面或操作完成后模块内等待刷新的加载场景。

<code src="./_example/text" />

## 文字和图标共同显示加载

加载过程中有文字和图标共同显示。适用于打开页面或操作完成后页面内等待刷新的加载场景。

<code src="./_example/icon-text" />

## 不同尺寸的加载
